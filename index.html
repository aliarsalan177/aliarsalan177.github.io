throttle.ai

Universal middleware that sits between AI coding agents and LLM providers to cut token usage 50–90% without losing context. A transparent local proxy — it speaks each provider's exact wire protocol — with prompt caching, context dedup, and diff-based file compression. Drops in with Claude Code, Cline, Continue, aider and more; local-first (binds 127.0.0.1), 158 tests, ~99% coverage.

  • TypeScript
  • LLM proxy
  • Token optimization
  • Prompt caching
  • Node ≥20
  • AI infra
Featured2026

Formwright

An LLM schema-driven, signal-reactive, framework-agnostic form engine. Define a form once as data — hand-written or AI-generated — and render it to surgically-updating DOM with no virtual DOM and no full re-renders. ~12 KB gzipped, zero dependencies, published as a suite of npm packages.

  • TypeScript
  • Signals
  • Zero-deps
  • npm packages
  • LLM schema
  • Web Components
